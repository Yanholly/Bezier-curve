# Bezier Curve 

Это вторая лабораторная работа курса **Компьютерная графика**. В ней 
был реализован алгоритм**кривой Безье N порядка**, а так же 
алгоритмы поворота вокруг осей X и Y.  Отрисовка происходит в реальном 
времени без использования **OpenGL библиотек**(по заданию).
